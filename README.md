# Low Society

**WARNING - NOT SAFE FOR WORK OR KIDS**

This is a re-theme of Reiner Knizia's
[High Society](https://boardgamegeek.com/boardgame/220/high-society) card
game.  All images and text are stolen for my friends' enjoyment.  Please go
buy a _real_ copy of the game and don't sue me.  See also
[this thread](https://boardgamegeek.com/thread/1485375).

The PDF includes printing instructios on page 5.  The main one is, don't let
your printer auto-scale the page; you want the cards to come out 2.5" x 3.5".
(Weirdly, my current printer changes the scale _slightly_ when printing in
"high quality" mode; did not figure out why.)  Cut them out & sleeve them; add
a standard playing card to each sleeve for bulk (but you already knew that).
